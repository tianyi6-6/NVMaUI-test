# 航迹信息记录器

基于PySide6开发的航迹信息记录UI应用程序，用于实验人员手工记录航迹信息。

## 功能特性

- **航迹记录控制**: 开始记录、记录中间点、结束记录
- **参数输入**: 航速、定深、距离、目标姿态、备注
- **实时显示**: 当前时间、记录状态、航迹点列表
- **数据保存**: 自动保存为CSV格式，包含时间戳
- **数据导出**: 支持手动导出航迹数据

## 安装要求

```bash
pip install PySide6
```

## 运行方法

### 方法1: 直接运行主程序
```bash
python trace_line_info_recorder.py
```

### 方法2: 使用启动脚本
```bash
python run_trace_recorder.py
```

## 使用说明

### 1. 开始记录航迹
1. 在"规划航迹参数"区域输入必要信息：
   - 航线规划航速 (m/s)
   - 航线规划定深 (m)
   - 航线规划距离 (m)
   - 目标姿态倾角（°）
   - 航迹航向角（°）
   - 线圈电流 (A)
2. 在"备注信息"区域输入相关备注
3. 点击"开始记录航迹"按钮

### 2. 记录中间点
- 在航迹记录过程中，可以点击"记录中间点"按钮记录航迹中的关键点
- 系统会先生成时间戳，然后弹出对话框输入信息，确保时间精度
- 需要输入该航点的距离（米）、航点序号、纬度和经度
- 距离输入支持小数点后2位精度
- 航点序号可以自定义，便于标识和管理
- **经纬度输入采用度分格式**：
  - 纬度：度(0-90) + 分(0.00000-59.99999) + 半球(N/S)
  - 经度：度(0-180) + 分(0.00000-59.99999) + 半球(E/W)
  - 示例：39°54.25200'N, 116°24.44400'E

### 3. 结束记录
- 点击"记录结束点"按钮完成航迹记录
- 系统会先生成时间戳，然后弹出对话框输入信息，确保时间精度
- 需要输入该航点的距离（米）和航点序号
- 距离输入支持小数点后2位精度
- 系统会自动保存航迹数据到指定目录

### 4. 数据管理
- **清空当前航迹**: 清除当前未保存的航迹数据
- **导出航迹数据**: 将当前航迹数据导出到指定位置

## 数据保存

### 保存位置
所有数据保存在 `D:/trace_data` 目录下

### 文件格式
每次记录会生成两个CSV文件：

1. **主航迹信息文件** (`trace_YYYYMMDD_HHMMSS.csv`)
   - 包含：起始时间戳、最近点时间戳、结束时间戳、航线规划航速、航线规划定深、航线规划距离、目标姿态倾角、航迹航向角、线圈电流、备注

2. **航迹点详细信息文件** (`trace_points_YYYYMMDD_HHMMSS.csv`)
   - 包含：序号、时间戳、类型、距离(m)、航点序号、纬度(十进制度)、经度(十进制度)、纬度(度分)、经度(度分)、备注

### 文件命名规则
- 使用航迹开始时间作为文件名
- 格式：`YYYYMMDD_HHMMSS` (年月日_时分秒)

## 界面说明

### 左侧控制面板
- **记录控制**: 开始、中间点、结束按钮
- **航迹参数**: 航速、定深、距离、目标姿态输入框
- **备注信息**: 多行文本输入框
- **记录状态**: 显示当前记录状态和实时时间

### 右侧数据显示面板
- **当前航迹信息**: 显示起始时间、最近点时间、结束时间、记录点数
- **航迹点列表**: 表格显示所有记录的航迹点（包含距离、航点序号和经纬度信息，经纬度以度分格式显示）
- **表格编辑**: 支持双击编辑距离、航点序号和备注，经纬度列不可编辑（需通过记录中间点重新输入）
- **操作按钮**: 清空和导出功能

## 注意事项

1. **必填字段**: 开始记录前必须填写航线规划航速、航线规划定深、航线规划距离、目标姿态倾角、航迹航向角、线圈电流
2. **记录状态**: 记录过程中参数输入框会被锁定，防止误操作
3. **数据安全**: 每次结束记录都会自动保存，确保数据不丢失
4. **文件编码**: 所有CSV文件使用UTF-8编码，支持中文
5. **时间精度**: 中间点和结束点会先生成时间戳再弹窗，确保时间精度
6. **表格编辑**: 双击表格单元格可编辑内容，编辑后自动保存到数据
7. **航点序号**: 起始点默认为"START"，其他航点可自定义序号
8. **经纬度记录**: 每个中间点都可以记录精确的经纬度坐标（度分格式，分精度到小数点后5位）

## 错误处理

- 如果未安装PySide6，程序会提示安装
- 如果数据目录无法创建，会显示错误信息
- 如果必填字段为空，会弹出警告对话框
