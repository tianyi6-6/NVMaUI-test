# 航迹信息记录器打包说明

## 快速打包

### 方法1: 使用批处理文件（推荐）
```bash
# 双击运行
build.bat
```

### 方法2: 使用Python脚本
```bash
python build_exe.py
```

### 方法3: 手动打包
```bash
# 安装依赖
pip install -r requirements.txt

# 直接使用PyInstaller
pyinstaller --onefile --windowed --name="航迹信息记录器" trace_line_info_recorder.py
```

## 打包配置说明

### 主要参数
- `--onefile`: 打包为单个可执行文件
- `--windowed`: 不显示控制台窗口
- `--name`: 指定可执行文件名称
- `--icon`: 指定图标文件（可选）
- `--clean`: 清理临时文件

### 包含的文件
- 主程序: `trace_line_info_recorder.py`
- 说明文档: `README_trace_recorder.md`
- 图标文件: `icon.ico`（自动生成）

## 输出文件

### 生成的文件
- `dist/航迹信息记录器.exe` - 主程序
- `dist/使用说明.txt` - 简单使用说明
- `build/` - 构建临时文件（可删除）

### 文件大小
- 预计大小: 50-100 MB
- 包含完整的Python运行环境和所有依赖

## 部署说明

### 单机部署
1. 将 `dist` 文件夹复制到目标机器
2. 双击 `航迹信息记录器.exe` 运行
3. 程序会自动创建 `D:/trace_data` 目录

### 网络部署
1. 将可执行文件放在网络共享位置
2. 用户通过网络访问运行
3. 数据仍保存在本地 `D:/trace_data`

## 注意事项

### 系统要求
- Windows 10/11 (64位)
- 至少 200MB 可用磁盘空间
- 不需要安装Python

### 首次运行
- 可能需要较长时间启动
- 可能会被杀毒软件误报（可添加白名单）
- 需要管理员权限创建数据目录

### 故障排除

#### 打包失败
```bash
# 清理缓存
rmdir /s build
rmdir /s dist
rm *.spec

# 重新安装依赖
pip uninstall pyinstaller
pip install pyinstaller
```

#### 运行失败
1. 检查是否有杀毒软件拦截
2. 尝试以管理员身份运行
3. 检查D盘是否有写入权限

#### 缺少依赖
```bash
# 安装所有依赖
pip install -r requirements.txt

# 或者单独安装
pip install PySide6 pyinstaller pillow
```

## 自定义配置

### 修改图标
1. 准备一个 `.ico` 格式的图标文件
2. 命名为 `icon.ico` 放在项目根目录
3. 重新运行打包脚本

### 修改程序名称
编辑 `build_exe.py` 中的 `--name` 参数：
```python
"--name=你的程序名称",
```

### 添加其他文件
在 `build_exe.py` 中添加 `--add-data` 参数：
```python
"--add-data=你的文件;.",
```

## 版本管理

### 版本号更新
在 `trace_line_info_recorder.py` 中修改：
```python
app.setApplicationVersion("1.1")  # 更新版本号
```

### 发布包
1. 运行打包脚本
2. 压缩 `dist` 文件夹
3. 命名为 `航迹信息记录器_v1.1.zip`

## 技术支持

如果遇到问题，请检查：
1. Python版本是否为3.8+
2. 是否安装了所有依赖
3. 是否有足够的磁盘空间
4. 防火墙/杀毒软件设置
