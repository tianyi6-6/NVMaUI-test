# 航迹采集系统

一个基于PySide6和matplotlib的航迹采集系统，支持实验站设置、航点采集、卫星地图显示和距离计算。

## 功能特性

### 1. 控制面板
- **实验站设置**: 输入实验站的经纬度坐标（度:分格式）
- **航迹采集控制**: 开启/停止航迹采集
- **数据保存**: 结束采集后自动保存数据到指定文件夹
- **状态显示**: 实时显示系统状态信息

### 2. 航点表格
- **实时记录**: 显示所有采集的航点信息
- **时间戳**: 自动记录每个航点的采集时间
- **坐标显示**: 显示航点的经纬度坐标
- **描述信息**: 支持为每个航点添加描述

### 3. 实验区地图
- **地图背景**: 显示地理坐标网格和海岸线
- **实验站标记**: 用黄色高亮点显示实验站位置
- **航点可视化**: 用红色点标注所有航点
- **距离计算**: 自动计算航点到实验站的距离
- **航迹线**: 显示航点之间的连线轨迹

## 安装依赖

```bash
pip install -r requirements_geo.txt
```

## 运行程序

```bash
python geo_sites_locator.py
```

## 使用说明

### 1. 设置实验站
1. 在控制面板中输入实验站名称
2. 输入纬度（度:分格式）
3. 输入经度（度:分格式）
4. 点击"设置实验站"按钮
5. 地图上会显示黄色的实验站标记

### 2. 开启航迹采集
1. 点击"开启航迹采集"按钮
2. 按钮变为"停止航迹采集"，状态显示"航迹采集中..."
3. "新增航点"按钮变为可用状态

### 3. 添加航点
1. 点击"新增航点"按钮
2. 输入纬度（度:整数，分:6位小数）
3. 输入经度（度:整数，分:6位小数）
4. 输入描述信息（可选）
5. 确认后航点会添加到表格和地图中

### 4. 结束采集并保存
1. 点击"结束航迹采集并保存"按钮
2. 选择保存目录
3. 数据自动保存为JSON格式文件
4. 文件名格式：`航迹数据_YYYYMMDD_HHMMSS.json`

## 数据格式

保存的JSON文件包含以下信息：

```json
{
  "实验站": {
```

## 界面布局

- **左侧面板**: 控制面板和航点表格
- **右侧面板**: 实验区地图显示
- **分割器**: 可调整左右面板的比例

## 技术特点

### 距离计算
使用Haversine公式计算地球表面两点间的距离：
```
a = sin²(Δφ/2) + cos(φ₁) × cos(φ₂) × sin²(Δλ/2)
c = 2 × atan2(√a, √(1−a))
d = R × c
```

### 地图显示
- 使用地理坐标网格系统
- 显示简化的海岸线轮廓
- 支持自动缩放和范围调整

### 坐标格式
- 支持度:分格式输入（如：39°54.252'）
- 内部转换为十进制度数存储
- 显示精度为6位小数

## 系统要求

- Python 3.8+
- PySide6 6.5.0+
- matplotlib 3.7.0+
- numpy 1.24.0+

## 注意事项

1. 经纬度输入使用度:分格式
2. 航点采集需要先设置实验站
3. 数据保存为UTF-8编码的JSON文件
4. 地图显示为简化版本，主要用于航点可视化
